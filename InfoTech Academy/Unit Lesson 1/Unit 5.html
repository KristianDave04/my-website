    UNIT V: Wireless Networking

    1. What is Wireless Networking?
    • Wireless networking enables devices to connect and communicate without physical
    cables, using radio frequency (RF) signals or infrared (IR) signals to transmit data. • This module explores how wireless networks function, the standards involved, security concerns, and setup basics.

    2. Key Components of a Wireless Network

    Component Description
    Wireless Access Point (WAP) Connects wireless devices to a wired network
    Wireless Router Combines a router and WAP; used in most homes
    Wireless Network Interface Card (NIC) Enables devices to connect wirelessly
    Antenna Used by access points and NICs to transmit/receive signals
    Signal Transmitted over 2.4 GHz or 5 GHz bands (or 6 GHz forWi-Fi 6E)

    3. Wireless Standards (IEEE 802.11)

    Standard Frequency Max Speed Range Use Case
    802.11a 5 GHz 54 Mbps Short Obsolete
    802.11b 2.4 GHz 11 Mbps Long Obsolete
    802.11g 2.4 GHz 54 Mbps Medium Legacy
    802.11n 2.4/5 GHz 600 Mbps Wide Common
    802.11ac 5 GHz 1.3 Gbps+ Medium High-speed networks
    802.11ax (Wi-Fi 6) 2.4/5 GHz 10 Gbps Efficient C Modern
    802.11be (Wi-Fi 7) 6 GHz 30 Gbps+ Future-ready

    4. Wireless Modes

    Mode Description
    Infrastructure Mode Devices connect through an access point (standard Wi-Fi setup)
    Ad Hoc Mode Devices connect directly without access points
    Mesh Networking Devices relay data to extend coverage (used in large areas)

    5. Wireless Frequency Bands
    * 2.4 GHz: Longer range, more interference (microwaves, Bluetooth)
    * 5 GHz: Shorter range, higher speeds, less interference
    * 6 GHz: Ultra-fast, used in Wi-Fi 6E and Wi-Fi 7 (newest standard)

    6. Wireless Security Protocols

    Protocol Description Status
    WEP Weak encryption Deprecated
    WPA Improved over WEP Legacy
    WPA2 AES-based, strong encryption Current standard
    WPA3 Enhanced security C forward secrecy Emerging standard

    7. Common Wireless Networking Terms

    Term Meaning
    SSID Service Set Identifier – network name
    BSSID MAC address of the access point
    Roaming Moving between access points without losing connection
    Channel Subdivision of frequency band to reduce interference
    Interference Signal disruption from overlapping Wi-Fi or devices

    8. Wireless Troubleshooting Tips
    * Weak signal: Move closer to the access point or use repeaters/extenders
    * Interference: Change channel on the router
    * Authentication errors: Check WPA settings and credentials
    * Device not connecting: Restart router, check NIC drivers

    9. Benefits and Challenges

    Advantages:
    * Mobility and convenience
    * Easier installation
    * Scalability

    Challenges:
    * Signal interference
    * Security risks
    * Limited bandwidth with many users

    10. Tools for Wireless Analysis
    * Wireshark: Monitor wireless traffic
    * NetSpot/ inSSIDer: View signal strength and channel overlap
    * Aircrack-ng: For ethical hacking and penetration testing.
