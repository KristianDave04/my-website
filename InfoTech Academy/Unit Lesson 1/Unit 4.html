    UNIT IV: Network Devices – Routers, Switches, and Hubs

    1. Hub
    Definition:
    A hub is a basic networking device that connects multiple devices in a LAN and forwards data to all ports, regardless of the intended recipient.

    Characteristics:

    Feature Description
    OSI Layer Layer 1 (Physical)
    Intelligence None – does not filter data
    Data Transmission Broadcasts to all devices
    Speed Typically 10/100 Mbps
    Full/Half Duplex Half-duplex only
    Use Case Very small or legacy networks

    2. Switch
    Definition:
    A switch is a network device that connects devices within a LAN and uses MAC addresses to intelligently forward data only to the appropriate destination device.

    Characteristics:

    Feature Description
    OSI Layer Layer 2 (Data Link)
    Intelligence Maintains a MAC address table
    Data Transmission Unicast to specific destination
    Speed 10/100/1000 Mbps and above
    Full/Half Duplex Full duplex supported
    Use Case Modern home and business networks

    Advanced Features:
    * VLAN Support: Segments network virtually
    * PoE (Power over Ethernet): Powers devices like IP cameras
    * Port Security: Limits devices per port
    * Managed vs Unmanaged: Managed switches allow configuration and monitoring

    3. Router
    Definition:
    A router connects multiple networks together (e.g., home network to the Internet) and routes data packets based on IP addresses.

    Characteristics:

    Feature Description
    OSI Layer Layer 3 (Network)
    Intelligence Uses routing tables and protocols
    Data Transmission Forwards packets between networks
    Speed Varies, often with built-in firewall and NAT
    Use Case Home internet, enterprise WANs, ISPs

    Key Functions:
    * Assign IP addresses via DHCP
    * Translate addresses using NAT
    * Filter traffic using ACLs
    * Route selection using static or dynamic routing protocols

    Types of Routers:
    * Core Router: High-speed, used in backbone networks
    * Edge Router: Connects internal networks to external ones (e.g., ISPs)
    * Wireless Router: Combines router and wireless access point (e.g., home routers)

    4. Comparative Summary

    Feature Hub Switch Router
    OSI Layer Layer 1 Layer 2 Layer 3
    Address Type None MAC Address IP Address
    Data Forwarding Broadcast Unicast/Broadcast/Multicast Route based on IP, Collision Domain
    Broadcast Domain One One (can segment with VLANs) Each interface is separate

    Intelligence Dumb Smart Smart + Routing

    5. Other Related Devices (Brief Overview)

    Device Description
    Modem Modulates and demodulates signals for Internet access via ISP
    Access Point Extends wireless coverage in a network
    Firewall Filters traffic based on security rules
    Bridge Connects and filters traffic between two LAN segments
