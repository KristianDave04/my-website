    UNIT I: Networking Basics and Terminology


    1. What is a Network?
    A computer network is a group of interconnected devices that communicate and share resources (such as files, internet access, printers) with each other.
    * Purpose: Resource sharing, communication, and centralized data access.
    * Examples: Office LAN, the Internet, school networks.



    2. Types of Networks
    * LAN (Local Area Network) – Small geographic area, such as homes, schools, or offices.
    * WAN (Wide Area Network) – Covers large areas like cities or countries (e.g., the Internet).
    * MAN (Metropolitan Area Network) – Larger than a LAN but smaller than a WAN, like a citywide network.
    * PAN (Personal Area Network) – Very small area, like connecting your phone to a laptop via Bluetooth.



    3. Network Topologies
    The physical or logical layout of network devices.
    * Bus – All devices connected to a single cable.
    * Star – Devices connected to a central hub or switch.
    * Ring – Each device connected to two others, forming a ring.
    * Mesh – Devices are interconnected, offering multiple paths.
    * Hybrid – Combination of two or more topologies.



    4. Communication Types
    * Unicast: One-to-one communication.
    * Broadcast: One-to-all (within a network segment).
    * Multicast: One-to-many (specific group).
    * Anycast: One-to-nearest (in terms of routing).



    5. Transmission Modes
    * Simplex: Data flows in one direction only (e.g., keyboard to PC).
    * Half-Duplex: Data flows in both directions, but only one at a time (e.g., walkie-talkies).
    * Full-Duplex: Data flows in both directions simultaneously (e.g., telephones).



    6. Key Networking Devices (Basic Introduction)
    * Hub – Basic device that sends data to all connected devices.
    * Switch – Smarter than a hub; sends data only to the intended device.
    * Router – Connects different networks; routes traffic.
    * Modem – Modulates and demodulates signals for Internet access.
    * NIC (Network Interface Card) – Hardware that connects a computer to a network.



    7. Bandwidth, Latency, and Throughput
    * Bandwidth: Maximum data transfer rate of a network (measured in Mbps/Gbps).
    * Latency: Time it takes for data to travel from source to destination.
    * Throughput: Actual rate of successful data delivery.



    8. Protocols (Basic Understanding)
    * Rules governing communication in a network.
    * Examples:
    * HTTP/HTTPS – Web browsing
    * FTP – File transfer
    * TCP/IP – Core suite for data transmission
    * DNS – Resolves domain names to IP addresses



    9. Network Models (Overview)
    * Client-Server: Central server provides resources to clients.
    * Peer-to-Peer (P2P): All nodes have equal status and share resources directly.



    10. Introduction to MAC and IP Addressing
    * MAC Address: Physical address, unique to each network interface.
    * IP Address: Logical address used to identify devices on a network.
