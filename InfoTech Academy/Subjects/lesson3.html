    • Can contain combinations of letters (lower-case & upper-case), digits or numbers, or underscore (_). (Ex. Var_1, my_char1)

    OperatorsNameExample

    OperatorsExampleSame as

    Comparison Operators: used for comparing values (used in conditionals)

    OperatorsNameExample

    !=Not Equalx != y
    >Greater thanx > y
    <Less thanx < y
    >=Greater or Equalx >= y

    Logical Operators: used to combine conditional statements

    OperatorsDescriptionExample

    Let's Try This!

    Activity Title: “Create Your First Python Mini App: A Simple Quiz!”

    Objective: Help beginners understand Python basics by building a small interactive quiz program.

    Instructions:
    Scenario: You’re designing a short Python quiz app that asks the user 3 questions. Collect answers, evaluate, and show score.

    1. Set Up Your Python Script (5 mins)
      Create a file quiz.py and print a welcome message.
    2. Ask Questions (10 mins)
      Use input() for 3 questions. Store responses.
    3. Check Answers (15 mins)
      Use if/else to compare with correct answers. Track score.
    4. Show Score (5 mins)
      Print total correct answers.
    5. Bonus (Optional)
      Use .lower(), add more questions or retries.

    Example Output:

    print("Welcome to the Python Quiz!")
q1 = input("What is the keyword to define a function in Python? ")
score = 0
if q1.lower() == "def":
    print("Correct!")
    score += 1
else:
    print("Oops! The correct answer is 'def'.")

    Deliverables:

    • A .py file with quiz code
    • Optional: Screenshot of terminal output

    Evaluation Criteria:

    CriteriaPoints
    Code runs without errors10
    Use of variables and input10
    Correct use of if/else10
    Output formatting10
    Bonus features10
    Total50

    Extension / Homework (Optional):

    • Ask for user name and personalize greeting
    • Show score percentage or feedback
    • Use a loop to retake the quiz
