    UNIT II: CSS Styling and Flex/Grid Layout

    Introduces Cascading Style Sheets (CSS) for designing visually appealing websites. Explains styling basics, selectors, colors, fonts, and modern layout techniques like Flexbox and CSS Grid for creating responsive and structured page layouts.


    UNIT III: Introduction to JavaScript

    Provides an overview of JavaScript fundamentals, including variables, data types, functions, control structures, and basic event handling. Enables interactivity and dynamic behavior in web pages.


    UNIT IV: Document Object Model (DOM)

    Explores how JavaScript interacts with HTML through the DOM. Covers selecting and modifying elements, handling events, and dynamically updating the web page content and structure in response to user actions.


    UNIT V: Responsive Web Design (RWD)

    Focuses on designing web pages that adapt to different screen sizes and devices. Topics include media queries, flexible layouts, responsive images, and mobile-first design principles.


    UNIT VI: Web Accessibility (a11y)

    Emphasizes making websites usable for everyone, including people with disabilities. Covers semantic HTML, ARIA roles, keyboard navigation, and best practices for inclusive design.


    UNIT VII: Deploying Websites: A Comprehensive Guide

    Explains how to publish a website to the internet. Includes domain names, hosting options, FTP/SFTP, GitHub Pages, and tools for version control and site optimization.


    Let's Try This!


    Activity Title: “Build a Personal Web Page Using HTML5!”

    Objective:
    Learners will understand and apply correct HTML5 structure and syntax by creating a basic personal web page using semantic elements.

    Instructions:
    Scenario:
    You're tasked with creating a simple personal profile web page using HTML5. The page should display your name, bio, photo, hobbies, and contact information using semantic HTML5 elements.

    Activity Tasks:

    1. Set Up Your HTML5 Document (5 mins)
    o Use the correct <!DOCTYPE html> declaration.
    o Add the basic structure: <html>, <head,> <title>, <body>.

    2.Add Semantic Sections (15 mins)
    Use the following HTML5 semantic elements to organize your content:
    o <header>: Page title and navigation (if applicable)
    o <main>: Main content area
    o <section>: Bio or hobbies
    o <article>: A short paragraph about yourself
    o <aside>: Fun facts or links
    o <footer>: Contact info (email or social media links)

    3. Embed Media and Links (10 mins)
    o Add an image (<img>) for your profile picture.
    o Use <a> tags to link to at least 2 social profiles or external sites.

    4.Validate Your Syntax (5 mins)
    o Make sure your HTML is properly nested and indented.
    o Check with W3C Validator if internet is available.

    5.Bonus Challenge (Optional - 10 mins)
    o Include an HTML5 <video> or <audio> element.
    o Use <form> to add a simple contact form (name, email, message).

    Deliverables:
    • A fully functional .html file.
    • Optional: Screenshot or screen share of your page preview.

    Evaluation Criteria:

    Criteria Points
    Correct HTML5 structure 10
    Proper use of semantic elements 15
    Syntax accuracy and indentation 10
    Inclusion of media/links 10
    Overall presentation 5
    Total 50

    Extension / Homework (Optional):
    • Research and explain the difference between:
    o <section> vs <div>
    o <article> vs <aside>
    • Modify the page with CSS to improve visual appearance (for the next class or bonus).
