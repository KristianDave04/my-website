    UNIT II: CSS Styling and Flex/Grid Layout

    1. Introduction to CSS (Cascading Style Sheets)
    What is CSS?
    * CSS is the language used to describe the presentation of HTML documents, including layout, design, and visual effects.
    *It allows you to controlthe color, font, spacing, positioning, and responsiveness ofthe elements in an HTML page

    Why use CSS?
    * Separation of content and style: CSS enables you to separate HTML structure from its presentation, making the web pages cleaner and easier to maintain.
    * Consistency: With CSS, you can apply the same styles across multiple pages, ensuring consistent design.

    2. CSS Syntax
    CSS follows a simple structure of selectors, properties, and values.

    H1 {
    color: blue;
    font-size: 32px;
    }

    * Selector: `h1` refers to all `<h1>` elements in the document.
    * Property: `color` is the property being styled.
    * Value: `blue` is the value being applied to the color property.

    3. Types of CSS
    Inline CSS:
    • Directly written inside the HTML element using the `style` attribute.

    <h1 style=”color: red;”>This is a Red Heading</h1>

    Internal CSS:
    • Written within `<style>` tags inside the `<head>` section of the document

    <head>
    <style>
    H1 {
    Color: green;
    }
    </style>
    </head>

    External CSS:
    • Stored in a separate `.css` file and linked to the HTML document.

    <link rel=”stylesheet” href=”styles.css”>

    4. CSS Selectors
    Selectors target HTML elements to apply styles. Some common types of selectors include:

    Selector Type Description
    Element Selector Targets elements by their tag name (`h1`, `p`)
    Class Selector Targets elements with a specific class (`.class-name`)
    ID Selector Targets an element with a specific `id` (`#id-name`)
    Universal Selector Targets all elements in the document (`*`)
    Attribute Selector Targets elements based on attributes (e.g., `[type=”text”]`)

    Examples

    /* Element selector*/
    H1 {
    font-size: 2rem;
    }

    /* Class selector */
    .button {
    background-color: blue;
    color: white;
    }

    /*ID selector*/
    #header {
    background-color: lightgray;
    }

    /* Attribute selector */
    input[type=”text”] {
    border: 2px solid gray;
    }

    5. CSS Box Model
    All HTML elements are viewed as rectangular boxes. The box model includes:

    1. Contentl: The actual content (e.g., text or images).
    2. Padding: Space between the content and the border.
    3. Border: Surrounds the padding (optional).
    4. Margin: Space outside the border between elements.

    * The `margin` creates space outside the box (20px).
    * The `padding` creates space inside the box (15px).
    * The `border` wraps around the padding and content (5px thick).

    6. Flexbox Layout

    What is Flexbox?
    * Flexbox (Flexible Box)is a one-dimensional layout modelfor distributing space between items in a container, either in a row or a column.
    *It helps align and distribute space within items, even when their sizes are unknown or dynamic.

    Key Flexbox Properties:
    For the container:
    * `display: flex` — Defines the container as a flex container.
    * `flex-direction: row | column` — Defines the direction of items (row or column).
    * `justify-content: flex-start | flex-end | center | space-between | space-around` — Aligns items along the main axis.
    * `align-items: flex-start | flex-end | center | stretch` — Aligns items along the cross axis.

    For the items:
    * `flex-grow` — Defines how much an item can grow relative to the other items.
    * `flex-shrink` — Defines how much an item can shrink if the container size is reduced.
    * `flex-basis` — Defines the initial size of an item before the available space is distributed.

    Example of Flexbox Layout:

    ```css

    .container {
    Display: flex;
    Justify-content: space-between; /* Distribute space between items */
    }

    .item {
    Width: 100px;
    Height: 100px;
    Background-color: lightblue;
    }
    ```     		```html

    <div class=”container”>
    <div class=”item”></div>
    <div class=”item”></div>
    <div class=”item”></div>
    </div>
    ```
