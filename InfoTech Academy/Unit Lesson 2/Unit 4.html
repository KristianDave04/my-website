    UNIT IV: Document Object Model (DOM)

    DOM (Document Object Model) Manipulation refers to the process of interacting with and modifying the structure, content, and style of a webpage using JavaScript. The DOM represents the webpage as a tree-like structure where each element, attribute, and piece of text in the HTML document is a node that can be accessed and manipulated.

    1. Understanding the DOM Structure:
    The DOM is a programming interface for web documents. It provides a structured representation ofthe HTML or XML document, allowing programming languages like JavaScript to manipulate the document’s structure, style, and content.

    * Document Node: Represents the entire HTML document.
    * Element Nodes: Represent HTML tags (e.g., `<div>`, `<p>`, `<a>`).
    * Text Nodes: Contain the text within an HTML element.
    * Attribute Nodes: Representthe attributes within an element(e.g., `id`, `class`, `src`).

    2. Accessing Elements in the DOM:
    To manipulate the DOM, you first need to access the elements. There are several ways to access DOM elements:
    • `getElementById(id)`: Returns the element with the specified `id`.
    • `getElementsByClassName(className)`: Returns a live HTMLCollection of elements with the specified class name.
    • `getElementsByTagName(tagName)`: Returns a live HTMLCollection of elements with the specified tag name.
    • `querySelector(selector)`: Returns the first element that matches a specified CSS selector.

    3. Manipulating Elements:
    Once you’ve accessed an element, you can manipulate it in various ways:
    Changing Content:
    • `textContent`: Modifies the text content of an element.
    • `innerHTML`: Modifies the HTML content inside an element (can also be used to insert HTML).

    Changing Attributes:
    • `setAttribute(name, value)`: Sets a new value for an attributes
    • `getAttribute(name)`: Gets the value of an attribute.

    Changing Style:
    • `style`: Modify inline styles directly.

    Class Manipulation:
    • `classList.add(className)`: Adds a class to an element.
    • `classList.toggle(className)`: Toggles a class on or off.

    4. Creating and Inserting New Elements:
    You can create new elements dynamically and insert them into the DOM.
    • `createElement(tagName)`: Creates a new HTML element.
    • `appendChild(node)`: Adds a node as the last child of the specified element.
    • `insertBefore(newNode, referenceNode)`: Inserts a new node before a reference node.
    • `insertAdjacentHTML(position, text)`: Inserts HTML content at a specified position relative to the element.

    5. Removing Elements:
    You can remove elements from the DOM:
    • `removeChild(node)`: Removes a child element from its parent.
    • `remove()`: Removes the element from the DOM entirely.

    6. Event Handling:
    DOM manipulation is often accompanied by event handling. You can add event listeners to elements to trigger actions when a userinteracts with the page.
    • `addEventListener(event, function)`: Adds an eventlistenerto an element.

    7. Example of DOM Manipulation:
    Here’s a simple example of DOM manipulation where a button click changes the content of a paragraph:

    <!DOCTYPE html>
    <html>
    <head>
    <title>DOM Manipulation Example</title>
    </head>
    <body>
    <p id="myParagraph">This is the original paragraph<./p>
    <button onclick="changeContent()">Click Me!</button>
    <script>
    function changeContent() {
    document.getElementById("myParagraph").innerHTML = "The content has been changed!";
    }
    </script>
    </body>
    </html>
