    UNIT III: Introduction to JavaScript

    1. What is JavaScript?
    * JavaScript (JS) is a high-level, dynamic programming language that is primarily used to create interactive effects within web browsers. It is one ofthe three core technologies of web development alongside HTML and CSS.
    * JavaScript enables client-side scripting to interact with HTML and CSS, allowing dynamic web content, manipulation of the Document Object Model (DOM), form validation, animations, and much more.
    * JavaScript is also used for server-side programming (using Node.js) and is part of fullstack development.

    Why JavaScript?
    * Interactivity: Makes websites interactive (e.g., dropdown menus, sliders, image galleries).
    * Versatility: Can be used both on the client-side and server-side (Node.js).
    * Community: JavaScript has a large and active community, with extensive documentation, frameworks (React, Angular, Vue), and libraries (jQuery, Lodash).

    2. Basic Structure of JavaScript
    Script Tags in HTML:
    • JavaScript code is typically written within `<script>` tags in an HTML file, either inside the `<head>` or just before the closing `</body>` tag.
    • Inline JavaScript: Writing JavaScript directly in HTML tags using the `onclick`, `onload`, or other event attributes.

    3. JavaScript Syntax Basics
    Statements and Semicolons:
    • JavaScript code consists of statements that tellthe browserto perform specific tasks. Each statement usually ends with a semicolon (`;`).

    Basic Data Types in JavaScript:
    1. String: Represents text.
    * Example: `”Hello World”`

    2. Number: Represents numeric values.
    * Example: `42`, `3.14`

    3. Boolean: Represents a value of `true` or `false`.

    4. Null: Represents an empty or non-existent value.
    * Example: `let x = null;`

    5. Undefined: A variable that has been declared but not assigned a value.
    * Example: `let y;` // y is undefined

    6. Object: A collection of key-value pairs (e.g., `let person = {name: “Alice”, age: 25};`).

    7. Array: A list-like structure.

    • Example: `let numbers = [1, 2, 3, 4];`

    4. Operators in JavaScript
    Arithmetic Operators
    • Used for performing mathematical calculations.

    Operator Description Example
    `+` Addition `3 + 2`
    `-` Subtraction `5 – 3`
    `*` Multiplication `4 * 2`
    `/` Division `8 / 2`
    `%` Modulus (Remainder) `5 % 2`

    Comparison Operators
    • Used to compare two values.

    Operator Description Example
    `==` Equalto (value comparison) `5 == 5` (true)
    `===` Strict equality (value and type comparison) `5 === “5”` (false)
    `!=` Not equalto (value comparison) `5 != 3` (true)
    `>` Greater than `5 > 3` (true)
    `<` Less than `3 < 5` (true)
