    UNIT I: HTML5 Structure and Syntax

    1. What is HTML5?
    * HTML5 is the latest version of the HTML standard used to structure web pages and applications.
    * It introduces new elements and attributes for better support of multimedia, graphics, and web application features.
    * HTML5 is designed to be lmore semanticl, mobile-friendly, and betterintegrated with modern web technologies (e.g., CSS3, JavaScript).

    2. Basic HTML5 Document Structure
    An HTML5 document begins with the declaration `<!DOCTYPE html>` and has the following fundamental structure:

    <!DOCTYPE html>
    <html lang=”en”>
    <head>
    <meta charset=”UTF-8”>
    <meta name=”viewport” content=”width=device-width, initial-scale=1.0”>
    <title>HTML5 Page</title>
    </head>
    <body>
    <h1>Welcome to HTML5!</h1>
    <p>This is a simple HTML5 document structure.</p>
    </body>
    </html>

    Key Elements:
    * `<!DOCTYPE html>`: Defines the document type and version of HTML (HTML5).
    * `<html>`: The root element that contains the entire document.
    * `<head>`: Contains meta-information about the document (e.g., title, character set, viewport settings).
    * `<meta charset=”UTF-8”>`: Specifies the character encoding (UTF-8 is the most common and supports a wide range of characters).
    * `<meta name=”viewport”>`: Ensures the page is optimized for mobile devices (responsive design)
    * `<title>`: Specifies the title ofthe webpage, shown in the browsertab.
    * `<body>`: Contains the content of the webpage that will be displayed to the user.

    3. Common HTML5 Tags and Their Purpose

    Tag Purpose
    `<html>`</td> Root element ofthe HTML document
    `<head>` Contains meta information,title, and links to styles/scripts
    `<title>` Defines the title ofthe document (appears in browser tab)
    `<body>` Contains the visible content of the webpage
    `<h1>`–`<h6>` Headings: `<h1>` is the most important, `<h6>` is the least
    `<p>` Paragraph
    `<a href="URL">` Hyperlink
    `<img src="URL">` Image (with `src` attribute to define the image source)
    `<ul>`, `<ol>`, `<li>` Lists: `<ul>` is unordered, `<ol>` is ordered, `<li>` is a list item
    `<div>` A generic container(used forlayout or styling)
    `<span>` A generic inline container(used for styling parts oftext)
    `<br>` Line break (creates a new line)
    `<hr>` Horizontalrule (creates a thematic break orline)

    4. HTML5 Semantic Elements
    HTML5 introduces semantic elements that improve the structure ofthe document, making it easier to read and understand for both humans and search engines.

    Tag Description
    `<header> Represents the introductory content or navigational section ofthe page
    `<nav>` Represents navigation links
    `<main>` Represents the main content of the document
    `<article>` Represents independent content, like blog posts, news articles
    `<section>` Represents a section ofrelated content
    `<aside>` Represents content that is related but separate from the main content (like sidebars)
    `<footer>` Represents the footer ofthe document or section, often containing . Copyright or contact information

    Example:

    <article>
    <header>
    <h1>Article Title</h1>
    </header>
    <section>
    <p>This is the main content of the article.</p>
    <meta name=”viewport” content=”width=device-width, initial-scale=1.0”>
    </section>
    <footer>
    <p>Written by Author Name</p>
    </footer>
    </article>
